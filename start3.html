# 基础类型

TypeScript 支持 与 javaScript 几乎相同的数据类型，还支持枚举类型

# 布尔值 boolean

最基本的数据类型就是 true/false 值，在大部分语言都用boolean 来表示 TypeScript 也不例外

 let isColor:boolean = true

# 数字类型 number

 let area:number = 10

# 字符串类型 string

  let name:string = 'string'

  let sayStr = `Hello${name}`

# 数组 Array

typescript 有两种方式来声明 数组数据类型

let list: number[] = [1,2,3]

let myFamily:Array<string> = ['Father','son']

# 元组 Tuple

元祖类型允许表示一个已知元素数量和类型的数组，各元素的类型不必相同。比如，你可以定义一对值分别为string 和 number 类型的元祖。

  let x : [number,string]

  x = [1,'1']

  x=['1',1]//error

当访问一个已知的元素，会得到正确的类型


let b :number = 1

b = b+x[0]

x[1].substr(1)

当你访问超过范围的元素时候会报错

# 枚举 enum

enum 类型是对javaScript 标准数据类型的一个补充。 像 C# 等其它语言一样，使用枚举类型可以为一组数值赋予友好的名字。

enum Color {Red,Green,Blue} 

let c:Color  = Color.Red
console.log(c) //0

默认情况下 从 0开始为元素编号。你也可以自己设置对应变量的值

enum Color {good=100,medium=50,bad=0} 

console.log(Color.good) //100

我还可以通过值来获取key值 比如我现在100分

enum Color {good=100,medium=50,bad=0} 

console.log(Color[100]) //good

# 任意 any

有的时候，我们想偷懒，还有用处理不同浏览器的兼容的时候会输入any，这样我们就可以愉快的像写js一样不受TypeScript,使用any 一时爽 。。。


  let val :any = ''

  val = []

  val = true

  val = 123

  let  valArray :any[]= [1,'1',true]

  valArray[2] = 3

# 空 void

void表示没有任何类型，当一个函数没有返回值的时候 你可以设置返回值为void


  function say():void{
    console.log('Hello World')
  }

void 类型只可以赋值给 undefined | null

# null 和 undefined


 let n:null = null||undefined

 let u:undefined = undefined||null

TypScript 中可以理解为null===undefined

# 从不 never

never类型表示的是那些用不存在的值类型。例如


function error(message: string): never {
  throw new Error(message)
}

# 对象 object

object 表示非原始类型，也就是除 number，string，boolean，symbol，null或undefined 之外的类型。

declare function create(o: object | null): void

create({ prop: 0 }) // OK
create(null) // OK
create(1) //error

# 类型断言 as

有时候你会遇到这样的情况，你会比 TypeScript 更了解某个值的详细信息。 通常这会发生在你清楚地知道一个实体具有比它现有类型更确切的类型。

通过类型断言这种方式可以告诉编译器，“相信我，我知道自己在干什么”。 类型断言好比其它语言里的类型转换，但是不进行特殊的数据检查和解构。 它没有运行时的影响，只是在编译阶段起作用。 TypeScript 会假设你，程序员，已经进行了必须的检查。

类型断言有两种形式。 其一是“尖括号”语法：

  
let someValue: any = 'this is a string'

let strLength: number = (<string>someValue).length

另外一个 as 语法

let someValue: any = 'this is a string'

let strLength: number = (someValue as string).length

在react中 只支持as 语法
